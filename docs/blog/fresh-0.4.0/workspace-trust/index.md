# Workspace Trust

Open a folder that can run code — a project manifest (`Cargo.toml`, `package.json`…), a build script, or a direnv/mise env — and Fresh raises a full-screen security prompt that names exactly what it found and lets you **Trust folder & Allow Tooling**, **Keep Restricted**, or **Block** everything. Trust is per-session, surfaced by a clickable `{trust}` element that now leads the status bar: Restricted runs your system tools (git, ripgrep, the system python) but blocks the project's own scripts, env activation, and language servers.
